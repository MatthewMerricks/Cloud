enum Ookii.Dialogs.Wpf.DownlevelTextMode

An enumeration that displays how the text in the CredentialDialog.MainInstruction and CredentialDialog.Content properties is displayed on a credential dialog in Windows XP.

Windows XP does not support the distinct visual style of the main instruction, so there is no visual difference between the text of the CredentialDialog.MainInstruction and CredentialDialog.Content properties. Depending on the scenario, you may wish to hide either the main instruction or the content text.

Enumerator:
MainInstructionAndContent 

The text of the CredentialDialog.MainInstruction and CredentialDialog.Content properties is concatenated together, separated by an empty line.

MainInstructionOnly 

Only the text of the CredentialDialog.MainInstruction property is shown.

ContentOnly 

Only the text of the CredentialDialog.Content property is shown.
