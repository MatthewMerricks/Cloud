enum CloudApiPublic.Static.TraceType : int

Types of Trace Log entries (flags); must match TraceLog.xsd

Enumerator
NotEnabled 

No general trace (errors are handled seperately)

Communication 

Communication tracing estimates the HTTP headers and body content during communication; when used without the AddAuthorization flag, authorization parameters will be exluded such as authentication tokens or user/pass

AddAuthorization 

AddAuthorization is only valid in conjunction with Communication (CommunicationIncludeAuthorization); Adding this will cause authorization parameters to appear in communication trace such as authentication tokens or user/pass (in plain text!!!)

CommunicationIncludeAuthorization 

See flags Communication and AddAuthorization tags for this combination flag

FileChangeFlow 

Logging the flow of FileChanges is extremely costly, use for development purposes only
