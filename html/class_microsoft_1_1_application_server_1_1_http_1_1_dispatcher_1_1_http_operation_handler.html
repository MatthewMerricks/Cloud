Detailed Description

An abstract base class used to create custom HttpOperationHandler instances that execute based on declared input and output HttpParameters.

abstract IEnumerable<HttpParameter> Microsoft.ApplicationServer.Http.Dispatcher.HttpOperationHandler.OnGetInputParameters ( )
protectedpure virtual

Implemented in a derived class to return the input HttpParameters that the HttpOperationHandler expects to be provided whenever the HttpOperationHandler.Handle method is called. The HttpParameters must be returned in the same order the the HttpOperationHandler.Handle method will expect them in the input object array.

OnGetInputParameters is only called once and the HttpParameters are cached in a read-only collection.

Returns
The input HttpParameters that the HttpOperationHandler expects.
