enum Microsoft.ApplicationServer.Common.Test.TestDataVariations

An flags enum that can be used to indicate different variations of a given TestData instance.

Enumerator:
AsInstance 

An individual instance of a given TestData type.

AsDerivedType 

An individual instance of a type that derives from a given TestData type.

AsKnownType 

An individual instance of a given TestData type that has a property value that is a known type of the declared property type.

AsNullable 

A Nullable<> instance of a given TestData type. Only applies to instances of ValueTypeTestData.

AsList 

An instance of a System.Collections.Generic.List<> of a given TestData type.

AsArray 

An instance of a array of the TestData type.

AsIEnumerable 

An instance of an System.Collections.Generic.IEnumerable<> of a given TestData type.

AsIQueryable 

An instance of an System.Linq.IQueryable<> of a given TestData type.

AsDataMember 

An instance of a DataContract type in which a given TestData type is a member.

AsXmlElementProperty 

An instance of a type in which a given TestData type is decorated with a System.Xml.Serialization.XmlElementAttribute.

AllSingleInstances 

All of the flags for single instance variations of a given TestData type.

AllCollections 

All of the flags for collection variations of a given TestData type.

AllProperties 

All of the flags for variations in which a given TestData type is a property on another type.

AllInterfaces 

All of the flags for interface collection variations of a given TestData type.

AllNonInterfaces 

All of the flags except for the interface collection variations of a given TestData type.

All 

All of the flags for all of the variations of a given TestData type.

enum Microsoft.ApplicationServer.Common.Test.UnitTestLevel
Enumerator:
None 

Test is largely empty or stubbed out. This test will not be validated. Tests not marked with UnitTestLevelAttribute default to this level.

NotReady 

Some tests are runnable but the test is not ready for check in. This test will show as "Inconclusive" if run in MSTest even if it passes basic level of verifications.

InProgress 

Most tests have been implemented and are runnable, but the test is yet not considered complete. This test will show as "Passed" in MSTest if it passes the basic levels of verification.

Complete 

Tests are complete and merit deeper verification. This test will show as "Pass" if run in MSTest only if it meets all the verification.
