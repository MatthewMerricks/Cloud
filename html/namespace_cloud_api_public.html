Detailed Description

Contains all public API functions of Cloud for Windows XP through Windows 8. See detailed description.

Primary components:
CLSync performs active sync of files and folders within a specified path to a syncbox in Cloud.
PushNotification.CLNotification allows manual subscription to push notifications and is used internally by CLSync .
REST.CLHttpRest allows manual access to Cloud communication methods for file queries, posting changes, and file transfer. It is used internally by CLSync .
EventMessageReceiver.EventMessageReceiver acts as a view model for status changes such as growls and sync status messages. It also exposes events for messages and status changes. CLSync fires event callbacks upon changes.
Support.CLTrace writes a message to log from a required format string and any object parameter arguments. It is used conditionally by other public API classes when instantiated with advanced settings for trace.
