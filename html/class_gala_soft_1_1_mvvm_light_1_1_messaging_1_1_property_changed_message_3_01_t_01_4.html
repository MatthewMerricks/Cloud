Detailed Description

Passes a string property name (PropertyName) and a generic value (OldValue and NewValue) to a recipient. This message type can be used to propagate a PropertyChanged event to a recipient using the messenging system.

Template Parameters
TThe type of the OldValue and NewValue property.
