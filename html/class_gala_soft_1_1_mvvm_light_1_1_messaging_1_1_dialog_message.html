Detailed Description

Use this class to send a message requesting to display a message box with features corresponding to this message's properties. The Callback property should be used to notify the message's sender about the user's choice in the message box. Typically, you can use this message typ's recipient will be an element of the View, and the sender will possibly be a ViewModel.
