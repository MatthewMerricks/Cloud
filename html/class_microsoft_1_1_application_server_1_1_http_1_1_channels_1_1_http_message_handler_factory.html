Detailed Description

Default HTTP message handler factory used by HttpMessageHandlerBindingElement for instantiating a set of HTTP message handler types using their default constructors. For more complex initialization scenarios, derive from HttpMessageHandlerFactory and override the HttpMessageHandlerFactory.OnCreate method.

Initializes a new instance of the HttpMessageHandlerFactory class given a set of HTTP message handler types to instantiate using their default constructors.

Parameters
handlersAn ordered list of HTTP message handler types to be invoked as part of an HttpMessageHandlerBindingElement instance. HTTP message handler types must derive from DelegatingChannel and have a public constructor taking exactly one argument of type HttpMessageChannel. The handlers are invoked in a buttom-up fashion. That is, the last entry is called first and the first entry is called last.
