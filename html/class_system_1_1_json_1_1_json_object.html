Detailed Description

A JsonObject is an unordered collection of zero or more key/value pairs.

A JsonObject is an unordered collection of zero or more key/value pairs, where each key is a String and each value is a System.Json.JsonValue, which can be a System.Json.JsonPrimitive, a System.Json.JsonArray, or a JsonObject.
