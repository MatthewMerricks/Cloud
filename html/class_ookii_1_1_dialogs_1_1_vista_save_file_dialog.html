Detailed Description

Prompts the user to select a location for saving a file.

This class will use the Vista style file dialog if possible, and automatically fall back to the old-style dialog on versions of Windows older than Vista.

As of .Net 3.5 and .Net 2.0 SP1, the regular System.Windows.Forms.FileDialog class will also use the new Vista style dialogs. However, certain options, such as settings System.Windows.Forms.OpenFileDialog.ShowReadOnly, still cause that class to revert to the old style dialogs. For this reason, this class is still provided. It is recommended that you use the System.Windows.Forms.SaveFileDialog class whenever possible.
