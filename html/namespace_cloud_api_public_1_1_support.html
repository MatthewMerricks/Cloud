Detailed Description

This is a set of classes meant to emulate a subset of Cocoa's Grand Central Dispatch (GCD). We support multiple dispatch queues. Tasks must take a single "in" argument and return void. The single argument may be an arbitrary collection object, so the task's function will need to unpack the collection in case multiple parameters are required.

Enumeration Type Documentation

enum CloudApiPublic.Support.DispatchActionType

The dispatch action type definitions. Async: The caller enqueues the action with the user state and continues executing. Sync: The caller enqueues the action with the user state, and waits for that action to complete. Note that the queue is FIFO, so many other Async or Sync actions may run before this action runs and completes. The caller blocks until its own action completes.
