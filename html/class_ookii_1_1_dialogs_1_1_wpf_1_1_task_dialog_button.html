string Ookii.Dialogs.Wpf.TaskDialogButton.CommandLinkNote
getset

Gets or sets the text of the note associated with a command link button.

The text of the note associated with a command link button.

This property applies only to buttons where the Type property is Ookii.Dialogs.Wpf.ButtonType.Custom. For other button types, it is ignored.

In addition, it is used only if the TaskDialog.ButtonStyle property is set to TaskDialogButtonStyle.CommandLinks or TaskDialogButtonStyle.CommandLinksNoIcon; otherwise, it is ignored.
