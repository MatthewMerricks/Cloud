Detailed Description

Base class that does basic unit test class validation.

Unit test classes should derive from this class and call the ValidateUnitTestClass to self-validate using MSTest.

void Microsoft.ApplicationServer.Common.Test.Framework.UnitTest.ValidateUnitTestClass ( )
inlineprotected

Validates the derived class adheres to a common set of conventions.

This method discovers and executes all the test methods in this current UnitTest base class. To add more validation, simply add more test methods to this base class. Please mark the test methods private so that they cannot be inherited by the derived tests.
